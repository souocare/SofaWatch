# SofaWatch

SofaWatch is a self-hosted companion application for tracking television shows and movies.

The project will provide:

- A Python FastAPI backend.
- A SQLite database by default.
- TMDB and TVDB metadata integrations.
- A React Native frontend powered by Expo.
- Native applications for iOS and Android.
- A responsive web application.
- Multi-user support.
- First-run administrator setup.
- Internationalization support.

## Project structure

```text
backend/   Python API and server-side logic
frontend/  React Native and Expo application
docs/      Project documentation
scripts/   Development and maintenance scripts
```

## Status 
The project is currently in its initial setup phase.
